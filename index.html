Didacticiel jQuery

Introduction

jQuery est une bibliothèque JavaScript rapide, compacte et riche. Il réalise des choses comme le parcours et la manipulation de documents HTML, la gestion d'évènements, l'animation, l'AJAX, d'une manière très simple et avec une API fonctionnant sur une multitude de navigateurs. Avec une combinaison de versatilité et d'extensibilité, jQuery a changé la façon d'écrire du JavaScript pour des millions de personnes.
